Privacy Policy

Last updated: November 16, 2025

This Privacy Policy describes how FIFA Ultimate Team Automation Tool ("we", "our") collects, uses, and protects the information of users ("you", "user") when you use our browser extension.

Types of Data Collected

Our extension is designed to help you buy and sell players in the market. To do this, we may collect and use technical data related to interactions with the market web pages, such as:

  • Data related to ongoing transactions and buying/selling operations
  • Technical information about the functioning of the active page
  • User actions within the extension (e.g., clicks, navigation)

We do not collect or store personally identifiable information such as name, email address, authentication data, financial data, location data, or browsing history.

Use of Data

The data collected is used solely to:

  • Improve the user experience when using the extension
  • Provide support functionalities for buying and selling players in the market
  • Ensure the proper operation and updating of the extension

We do not share, sell, or transfer collected data to third parties.

Data Security

We implement appropriate technical and organizational measures to protect collected data from unauthorized access, loss, or accidental modification.

Use of Permissions and Data

The extension requires some limited and necessary access permissions for its operation, such as the ability to read and modify content on market pages, store temporary data, and interact with the active browser tab.

Remote Code

We do not use remote code. All code is included in the extension package.

User Rights

Since we do not collect personal data, there are no data to modify or delete. If you have questions regarding data usage, you can contact us at: lorenzotabarelli@gmail.com

Changes to this Privacy Policy
